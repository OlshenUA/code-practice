## This repository consists of different practice solutions from platforms like NeetCode, LeetCode and Exercism
It was made for automatic backups from those websites as well as a portfolio repo.
NeetCode syncs problem solutions into a folder depending on the subject, like *Data Structures & Algorithms*. Exercism uses the *solutions* folder. LeetCode files are uploaded manually into the corresponding *Leetcode* folder
